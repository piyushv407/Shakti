# Shakti Education - Landing Page

Welcome to the **Shakti Education** repository! This is a modern, responsive landing page designed for an online education platform.

## 🚀 Features

- **Modern & Premium Design**: Professional layout featuring vibrant gradients, glassmorphism, and beautiful micro-animations to create a visually striking first impression.
- **Fully Responsive**: Flawless experience across desktops, tablets, and mobile devices using CSS Grid and Flexbox.
- **Dynamic Interactions**: Includes smooth scrolling, hover effects, floating shapes animations, and scroll-triggered animations.
- **Structured Sections**:
  - Top Bar & Navigation with Mobile Menu
  - Engaging Hero Section
  - Course Categories
  - Achievement Statistics
  - Live Classes Showcase with Add to Cart functionality
  - Testimonials
  - Instagram Social Feed
- **SEO Optimized**: Semantic HTML5 structure with proper landmarks and aria-labels for accessibility.

## 🛠️ Technology Stack

- **HTML5**: Semantic tags, accessibility attributes (aria-labels, roles).
- **Vanilla CSS3**: 
  - CSS Variables for consistent theming
  - Flexbox and Grid for modern layouts
  - Keyframe animations and transitions
  - Media queries for responsive design
- **Vanilla JavaScript**: Mobile menu toggle, intersection observer for scroll animations, and smooth scrolling capabilities.
- **FontAwesome (v6.4.0)**: Modern vector icons.
- **Google Fonts**: Utilizes `Poppins` and `Urbanist` for beautiful typography.

## 💻 Getting Started

This is a static website, so running it locally is extremely simple.

### Prerequisites

No build tools or package managers are required! Any modern web browser will work.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/shakti-education.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd shakti-education
   ```
3. **Open the project:**
   Simply double-click on the `Shakti.html` file, or use a local development server like VS Code's "Live Server" extension for the best experience.
